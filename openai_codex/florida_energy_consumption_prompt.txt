Build a single self-contained HTML project called "State of Florida Data Heatmaps" that renders an interactive Florida county choropleth map with a left-hand navigation panel, a main map panel, and a full-width footer panel.

Project requirements:

Create a Python generator script that reads local Excel workbooks and outputs one standalone HTML file with all data embedded directly in the page. The generated HTML must not depend on any external JS or CSS runtime libraries. The only network fetch allowed during generation is for Florida county boundary GeoJSON.

Workspace files:
- florida_customer_outage.xlsx
- florida_energy_consumption.xlsx
- florida_ng_consumption.xlsx
- output HTML: florida_county_outage_map.html
- generator script: scripts/generate_florida_outage_map.py

Overall page behavior:
- Single-file HTML output
- Florida county choropleth heatmap
- Left navigation panel for layer controls
- Map supports zoom in, zoom out, reset view, mouse wheel zoom, and drag-to-pan
- Hover tooltip only, no click-to-pin details
- Full-width footer panel beneath both the left nav and map panel for source citations
- Responsive enough to work on desktop and smaller screens

Data layers:
1. Electrical - Percent Outage
2. Electrical - Energy Consumption
3. Natural Gas - Consumption

Layer behavior:
- Only one layer is active at a time
- The active layer control should behave like a radio-style toggle that can also be turned off by clicking the active item again
- When heatmap is off, counties return to a neutral fill
- Active Layer Summary and Legend update when the active layer changes
- Tooltip must show only the data relevant to the active layer
- For outage layer, tooltip should show:
  - county name
  - tracked customers
  - customers out
  - percent outage
- For electric energy layer, tooltip should show:
  - county name
  - selected year energy consumption
- For natural gas layer, tooltip should show:
  - county name
  - selected year natural gas consumption

Year selector behavior:
- Place a year dropdown directly beneath the Data Layer controls
- Show it only for the electric energy and natural gas layers
- Hide it for outage layer
- Changing the year must refresh:
  - county colors
  - active layer summary
  - legend values
  - tooltip values
- Workbook years run from 2017 to 2036
- Keep separate selected years for electric and natural gas

Data handling:
- Match counties by county name
- Normalize county naming mismatches such as DeSoto -> Desoto
- Outage workbook data may contain duplicated numeric digits; normalize malformed values before computing percent outage
- Electric workbook source field is elec
- Natural gas workbook source field is ng
- Aggregate energy and gas consumption by county and year using Consumption MMBtu
- Embed all needed data into the HTML

- Map panel shadow: 0 10px 20px #1F6884

Legends:
- Outage layer palette:
  - #FFF8DC
  - #f1df9a

Map framing:
- Default zoom should be slightly zoomed in relative to full-state fit
- Reset view should return to the saved default framing
- Use tuned default offsets so Florida sits left-shifted and slightly lower in the panel rather than perfectly centered
